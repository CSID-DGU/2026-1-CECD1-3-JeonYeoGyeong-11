# Line Budget Allowlist

## 목적

큰 파일의 net line growth를 관리한다.

Measurement:

```text
git diff --numstat <gate-start-sha> -- <protected-paths>
```

Rule:

```text
added - removed <= 0
```

## Exception Format

```text
path | expires_on | reason
```

현재 active exception은 없다.

## Protected Paths

```text
graphfl_lab/experiments/suites/vision/variants.py
graphfl_lab/strategies/graphfl/strategy.py
graphfl_lab/experiments/vision/suite.py
```

Canonical:

- `docs/maintenance/migration-and-compatibility.md`
