# Early Experiment Results Archive

## 목적

현재 evidence pack 이전의 raw update graph와 spectral-only 실험 결과를 요약한다.

## 결과 Summary

| 관찰 | 의미 |
|---|---|
| raw update graph gain 약함 | graph source와 topology 재설계 필요 |
| random control 경쟁력 확인 | matched control 비교 필요 |
| seed별 variation 확인 | multi-seed validation 필요 |
| diagnostic metric 필요 | accuracy 단일 비교에서 attribution framework로 전환 |
