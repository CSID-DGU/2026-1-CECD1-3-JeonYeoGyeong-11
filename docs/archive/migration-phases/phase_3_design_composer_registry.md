# Phase 3 GraphFLDesign Composer And Registry

## 목적

Method profile을 `GraphFLDesign`과 registry로 표현했다.

## 범위

| 영역 | 내용 |
|---|---|
| design metadata | lifecycle slot 조합 |
| registry | preset lookup |
| prior-work proxy | FedAMP, SFL, pFedGraph 계열 profile |
| CLI bridge | `graph_method`, `graph_source`, `graph_mode`, `aggregation_target` 연결 |

## 결과

| Result | 의미 |
|---|---|
| design registry 도입 | method를 component 조합으로 표현 |
| proxy support level 기록 | paper mechanism boundary 명시 |
| Phase 4 준비 | state/relation/topology module로 연결 |
