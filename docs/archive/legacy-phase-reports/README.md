# Legacy Phase Reports

## 목적

이 폴더는 현재 Graph-FL evidence claim 이전의 실험 기록을 보관한다.

## 리포트 지도

| 문서 | 기록 |
|---|---|
| `PHASE1_DIAGNOSTICS_REPORT.md` | diagnostic metric 초기 신호 |
| `PHASE2_GRAPH_INFORMATIVENESS_REPORT.md` | update graph informativeness 초기 평가 |
| `PHASE2_GRAPH_INFORMATIVENESS_*` | Phase 2 setting별 반복 평가 |
| `PHASE2_GRAPH_SOURCE_SANITY_REPORT_BATCH1.md` | graph source sanity check |
| `PHASE2_5_SMOOTHING_FAILURE_ANALYSIS.md` | smoothing-only path 분석 |
| `PHASE3_DOMINANCE_AWARE_REPORT.md` | dominance-aware baseline 분석 |

## 현재 연결

현재 claim과 evidence는 `docs/framework/` 문서를 기준으로 한다.
